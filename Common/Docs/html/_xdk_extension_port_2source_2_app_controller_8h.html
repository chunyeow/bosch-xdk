#define EXTENSION_PORT_GPIO_FEATURE   1

EXTENSION_PORT_GPIO_FEATURE is the GPIO feature control macro for extension port. If enabled (set as 1) then

  • The PA0 GPIO pin is programmed for generating interrupt on both raising edge and falling edge.
  • The PA1 GPIO Pin is configured as output and toggled the of the pin for a couple of times If disabled (set as 0) then this feature will not be built into the software.
#define EXTENSION_PORT_I2C_FEATURE   1

EXTENSION_PORT_I2C_FEATURE is the I2C feature control macro for extension port. If enabled (set as 1) then basic read and write to the interfaced device registers is performed. If disabled (set as 0) then this feature will not be built into the software.

#define EXTENSION_PORT_SPI_FEATURE   1

EXTENSION_PORT_SPI_FEATURE is the SPI feature control macro for extension port. If enabled (set as 1) then basic read and write to the interfaced device registers is performed. If disabled (set as 0) then this feature will not be built into the software.

#define EXTENSION_PORT_UART_FEATURE   1

EXTENSION_PORT_UART_FEATURE is the UART feature control macro for extension port. If enabled (set as 1) then whatever data received in RX of UART is echoed (transmitted) back to the TX of UART. If disabled (set as 0) then this feature will not be built into the software.
