Detailed Description

The main purpose of this interface is for XDK users to configure and use the XDK extension board effectively. This module provides ease of use API's for the novice programmer to start prototyping with XDK extension board. Currently the support is provided to configure the extension board I/O pins as GPIO, also providing support to configure and establish serial communication namely UART, SPI and I2C.
