Detailed Description

This file holds static configurations which are meant to be assigned to a peripheral at the XDK110 extension board at a certain point in time for the Initialize, Connect, Enable, Disable, Disconnect functions. It is preferred to have only macro definitions in this file so that it contains compile time necessary information only.
