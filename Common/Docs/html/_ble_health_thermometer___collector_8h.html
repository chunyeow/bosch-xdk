#define BLEHEALTHTHERMOMETER_SUPPORT_COLLECTOR   0

BLEHEALTHTHERMOMETER_SUPPORT_COLLECTOR Defines if the BLE HEALTH THERMOMETER Profile implementation supports the COLLECTOR Role If enabled ( set to 1 ) it enables all the following BLE HEALTH THERMOMETER Profile COLLECTOR configurations and Application Interfaces. The default value for this option is disabled (0).
