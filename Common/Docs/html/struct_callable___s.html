CallableFunc_T func
 

Detailed Description

The data structure for callable elements.

This data structure is used to hold a function pointer allowing it to be scheduled as a scheduler job or to be used as a callback function. It is highly recommended not to directly access the member of this data structure. Instead the functions Callable_assign(), Callable_isAssigned(), Callable_callback(), Callable_call(), Callable_clear() should be used.
