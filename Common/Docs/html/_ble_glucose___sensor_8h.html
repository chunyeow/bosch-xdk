#define BLEGLUCOSE_SUPPORT_SENSOR   0

BLEGLUCOSE_SUPPORT_SENSOR Defines if the BLE GLUCOSE SENSOR Profile implementation supports the SENSOR Role. If enabled ( set to 1 ) it enables all the following BLE GLUCOSE SENSOR Profile SENSOR configurations and Application Interfaces. The default value for this option is disabled (0).
