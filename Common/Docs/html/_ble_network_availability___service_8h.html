#define BLE_SUPPORT_NETWORKAVAILABILITY_SERVICE   0

BLE_SUPPORT_NETWORKAVAILABILITY_SERVICE Defines if the BLE local device enables access to the local Network Availability service and characteristic.

If enabled ( set to 1 ) it enables Profiles to get access to the network Availability Service and characteristic

The default value for this option is disabled (0).
