enum BSP_InterruptIndex_E

Enumeration used to report the interrupt that has occurred. In addition to the communication interface interrupts the sensor node could support peripherals that have IO interrupts e.g. in case of threshold the interrupt could be used to wakeup the MCU the BSP informs the application that the given peripheral has generated the interrupt indexed by INTERRUPT_INDEX_x.
