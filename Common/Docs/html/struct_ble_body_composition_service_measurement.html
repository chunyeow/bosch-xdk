Detailed Description

BleBodyCompositionServiceMeasurement type Define the current Body Composition Measurement value to be set during the call of BLEBODYCOMPOSITIONSERVICE_SetBodyCompositionMeasurement

Field Documentation

U16 basalMetabolismValue

If the BASAL METABOLISM feature is supported, it shall be provided and valid during each call of BLEBODYCOMPOSITIONSERVICE_SetBodyCompositionMeasurement Unit : kilo Joules with resolution of 1

U16 bodyFatPercentageValue

Body Fat Percentage field: The Body Fat Percentage field shall be included in the Body Composition Measurement characteristic.

BleBodyCompositionServiceMeasurementFlags flagValue

flagValue: The flags indicating what values are included in the measurement

U32 muscleMassValue

If the MUSCLE MASS feature is supported, it shall be provided and valid during each call of BLEBODYCOMPOSITIONSERVICE_SetBodyCompositionMeasurement Unit : when Measurement Units (bit 0 of Flag) set to 1: Imperial => Value is given in millipound when Measurement Units (bit 0 of Flag) set to 0 = SI => Value is given in grams

U16 musclePercentageValue

If the MUSCLE PERCENTAGE feature is supported, it shall be provided and valid during each call of BLEBODYCOMPOSITIONSERVICE_SetBodyCompositionMeasurement Unit : Percent with resolution of 0.1
